# Minecraft Web
Minecraft Web is a recreation of a popular game, Minecraft. IDK, Why even I wrote a popular game, because I know, most of you guys have player that Billion Dollar Game, but yah.. Imagine running that on a browser no less? Today's Browsers are really popular. They have the power to use your GPU and make shaders and make it work

The main premise of the project is built on Foundational Work and Optimization. Thanks to Vercidium, who has been my inspiration for making this Voxel World into an Optimized game.

I don't really know, what much to write so here are the controls:-

Key W: Front
Key A:  Left
Key S: Back
Key D: Right

Space: Up
Left Shift: Down

Double Space to Fly (Also Disables Collision so be aware and use carefully)

Press F3 for Debug tools like seeing Chunks Rendered, Entities Renedered and Chunk Co-ordinates
